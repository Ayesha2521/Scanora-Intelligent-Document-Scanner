# Scanora – Intelligent Document Scanner

Scanora is a Flutter and Firebase-powered intelligent document scanning application that uses OCR (Optical Character Recognition) to extract text from scanned documents. The application enables users to scan, categorize, search, store, and synchronize documents efficiently across devices.

## Features

* Document Scanning using Camera
* OCR Text Extraction
* Smart Document Categorization
* Searchable Document History
* Offline Storage with Hive
* Cloud Synchronization with Firebase
* Dark & Light Theme Support
* Secure Document Management

## Technologies Used

* Flutter
* Dart
* Firebase
* Cloud Firestore
* Google ML Kit OCR
* Hive Database
* Riverpod State Management
